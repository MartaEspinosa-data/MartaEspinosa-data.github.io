Hi. I'm Marta Espinosa

Portfolio

A sample of my work and personal projects.

Deforestation exploration

In this project you can observe the countries with the largest area of ​​deforestation using SQL to create awareness about the impact on the environment.

A Social News Aggregator

Udiddit is a social news aggregator. The duties for this project were fix all the issues with the data model, make the system more robust and adding some web analytics using SQL.

Movies Correlation

In this project the different variables that can affect the revenue of a movie are analyzed to find out if there is a correlation between them using PYTHON, PANDAS, NUMPY and SEABORN.

Brewing Company

Power BI report for Seven Sages Brewing Company. This data model makes possible for the company's CFO to review and analyze what beers sell well and which ones generate the highest profitability.

Covid project

In this project, the total cases and total deaths caused by the Covid-19 in the different countries and continents, are analyzed using PYTHON, ,NUMPY, PANDAS library and MATPLOTLIB.

Pet Device Report

Waggle is a startup that makes smart devices for pets. This POWER BI report compares the data from the dog and cat devices to see which one is more successful among consumers.

Market analysis Report

An online clothing chain needs to create a targeted marketing campaign. This POWER BI analysis and report help to determine the best product to advertise.
